# 📚 BookCloner

**اپلیکیشن خودکار برای تبدیل کتاب‌های دیجیتال (صرف نمایش) به PDF**

---

## 🎯 مشکل

زمانی که کتابی را خریداری می‌کنید و صرفاً دسترسی به نمایش آن دارید (بدون نسخه PDF)، نمی‌توانید آن را برای مطالعه آفلاین یا چاپ ذخیره کنید.

## ✨ حل

**BookCloner** این فرآیند را خودکار می‌کند:
- صفحات کتاب را خودکار اسکرین‌شات می‌گیرد
- هر صفحه را شماره‌گذاری می‌کند
- تصاویر را در پوشه‌ای منسجم ذخیره می‌کند
- سپس می‌توان با ابزارهای دیگر به PDF تبدیل کرد

---

## 🚀 نحوه استفاده

### ۱️⃣ نصب وابستگی‌ها
```bash
pip install pillow
```

### ۲️⃣ اجرای برنامه
```bash
python app.py
```

### ۳️⃣ مراحل کار

#### مرحله ۱: انتخاب محدوده صفحه
- برنامه را اجرا کنید
- کتاب را در صفحه‌نمایش قرار دهید
- **کلیک چپ و بکشید** تا محدوده صفحه کتاب را انتخاب کنید
- رنگ قرمز ناحیه انتخاب شده را نشان می‌دهد

#### مرحله ۲: ثبت نقطه کلیک
- **کلیک راست** در جایی که دکمه "صفحه بعدی" یا ناحیه تعامل است
- نقطه کلیک ذخیره می‌شود (مختصات باید را ببینید)

#### مرحله ۳: شروع حالت خودکار
- کلیک بر روی دکمه **"شروع اتومات (F3)"** یا فشار دهید `F3`
- برنامه:
  - ✅ اسکرین‌شات می‌گیرد
  - ✅ کلیک می‌کند برای صفحه بعدی
  - ✅ منتظر بارگذاری صفحه می‌ماند
  - ✅ دوباره تکرار می‌کند

#### متوقف کردن
- فشار دهید `ESC` برای توقف حالت خودکار
- یا بستن پنجره برای خروج

---

## ⌨️ میانبرهای صفحه‌کلید

| کلید | عملکرد |
|------|--------|
| `F1` | اسکرین‌شات دستی یک بار |
| `F2` | مینیمایز پنجره |
| `F3` | شروع/توقف حالت خودکار |
| `C` | پاک کردن انتخاب |
| `ESC` | توقف حالت خودکار |

---

## 🖱️ کنترل‌ها

- **کلیک چپ + بکشید**: انتخاب محدوده صفحه
- **کلیک راست**: ثبت نقطه کلیک (محل دکمه "صفحه بعدی")
- **دکمه مینیمایز**: مخفی کردن برنامه برای دیدن بهتر صفحه

## 🔄 تبدیل به PDF

پس از اتمام اسکرین‌شات‌گیری، می‌توانید از ابزارهای زیر استفاده کنید:

### گزینه ۱: ImageMagick
```bash
magick convert "صفحه_*.png" output.pdf
```

### گزینه ۲: ffmpeg
```bash
ffmpeg -framerate 1 -pattern_type glob -i '*.png' -c:v libx264 -pix_fmt yuv420p output.mp4
```

### گزینه ۳: Python PIL
```python
from PIL import Image
import os

images = sorted([f for f in os.listdir('D:/screens') if f.endswith('.png')])
image_list = [Image.open(f'D:/screens/{img}').convert('RGB') for img in images]

if image_list:
    image_list[0].save('output.pdf', save_all=True, append_images=image_list[1:])
```

## 💡 نکات مهم

1. **تاخیر بین صفحات**: در حالت خودکار `۱.۵ ثانیه` طول می‌کشد تا صفحه بارگذاری شود. اگر کند است، این مقدار را در کد تغییر دهید.

2. **مختصات دقیق**: اطمینان حاصل کنید نقطه کلیک دقیقاً روی دکمه/ناحیه است.

3. **رزولوشن صفحه**: محدوده انتخاب‌شده باید تا حدی ثابت باشد (صفحات متحرک نشود).

4. **نام‌گذاری فارسی**: اسامی فایل‌ها به فارسی است؛ در صورت مشکل، کد را تغییر دهید.

## 🐛 رفع‌مسائل

**مشکل**: برنامه کلیک نمی‌کند
- ✓ مطمئن شوید Windows حالت "دسترسی کامل" را برای Python اجازه داده است

**مشکل**: صفحات خالی ذخیره می‌شوند
- ✓ محدوده انتخاب‌شده را بازبینی کنید (باید کل صفحه را شامل شود)

- ✓ تاخیر بین صفحات را افزایش دهید (در کد `time.sleep(1.5)` را تغییر دهید)
